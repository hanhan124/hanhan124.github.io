Hugo 自动化部署到 GitHub Pages 详细指南

📌 简介

本教程将详细介绍如何使用 Hugo 构建个人博客，并通过 GitHub Actions 实现自动化部署到 GitHub Pages。整个流程包括：

  1. 安装 Hugo
  2. 创建 GitHub 仓库（一个私有仓库存储源码，一个公开仓库存储 public 文件）
  3. 配置 SSH 访问 GitHub
  4. 备份 Hugo 源码到 GitHub（私有仓库）
  5. 配置 GitHub Actions 实现自动化部署
  6. 绑定自定义域名（可选）
  7. Hugo 站点的测试与更新
  8. 新建博文的流程

1️⃣ 安装 Hugo

1.1 检查 Hugo 是否已安装

在终端运行以下命令检查 Hugo 版本：

1

hugo version

如果未安装，可使用 Homebrew 进行安装（适用于 macOS）：

1

brew install hugo

对于 Windows 用户，可以通过 Chocolatey 安装：

1

choco install hugo-extended

2️⃣ 创建 GitHub 仓库

2.1 创建两个 GitHub 仓库

  • 私有仓库（hugo-source）：存放 Hugo 源代码。
  • 公开仓库（hanhan124.github.io）：存放生成的静态网站文件（public 目录）。

步骤：

  1. 进入 GitHub
  2. 创建 hugo-source（设为私有仓库）
  3. 创建 hanhan124.github.io（设为公开仓库）

3️⃣ 配置 SSH 访问 GitHub

3.1 生成 SSH 密钥

在终端运行以下命令生成 SSH 密钥：

1

ssh-keygen -t ed25519 -C "your_email@example.com"

3.2 添加 SSH 公钥到 GitHub

获取 SSH 公钥：

1

cat ~/.ssh/id_ed25519.pub

复制公钥内容，并按照以下步骤添加到 GitHub：

  1. 进入 GitHub SSH Keys
  2. 点击 New SSH Key
  3. 粘贴 SSH 公钥，点击 Add SSH Key

3.3 测试 SSH 连接 

1

ssh -T git@github.com

5️⃣ 配置 GitHub Actions 实现自动化部署

5.1 生成 Personal Access Token（PAT）

  1. 进入 GitHub Token 生成页面
  2. 选择 Generate new token (classic)
  3. 勾选 repo 和 workflow 权限
  4. 生成 Token 并保存

5.2 在 hugo-source 仓库添加 GitHub Secret

  1. 进入 hugo-source 仓库
  2. 点击 Settings > Secrets and variables > Actions
  3. 点击 New repository secret
  4. 名称: PERSONAL_ACCESS_TOKEN
  5. 值: 粘贴 PAT 并保存

5.3 配置 GitHub Actions 自动部署

name: Deploy Hugo to GitHub Pages

on:
  push:
    branches:
      - main

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout source code
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Setup Hugo
        uses: peaceiris/actions-hugo@v2
        with:
          hugo-version: 'latest'
          extended: true

      - name: Ensure FixIt theme exists
        run: |
          if [ ! -d "themes/FixIt" ]; then
            echo "Error: FixIt theme is missing!"
            exit 1
          fi

      - name: Build site
        run: hugo --minify

      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          personal_token: ${{ secrets.PERSONAL_ACCESS_TOKEN }}
          publish_dir: ./public
          external_repository: hanhan124/hanhan124.github.io
          publish_branch: main

结论

  • hugo-source 私有仓库存放 Hugo 源代码。
  • hanhan124.github.io 公开仓库存放 public 目录。
  • 使用 GitHub Actions 自动化部署。
